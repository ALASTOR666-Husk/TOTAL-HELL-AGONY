------------Циклы while и for
зачастую встает задача сделать одно и тоже действие много раз, например - вывести товары из списка 1 за другим или перебрать числа от 1 до 10 и для каждого выполнить одинаковый код (сделать счетчик)
цикл while: первое - этот цикл проверяет блок кода пока указанное условие истинно, , если условие истинно то он выполняется. После выполнения блока кода условие снова проверяется, он повторяется пока не станет ложным

пример :

let i = 0
while(i < 5){
    alert(i)
    i= i+1
}

do while - позволяет выполнить 1 операцию и после этого начнется проверка цикла while

let i = 0
do {
    alert(i)
    i= i+1
}
while(i < 0)

for - состоит из 3 основных частей:
1 - инициализация (let i = 0)
2 - условие (i < 5)
3 - выражения обновления (i++)

пример:

let i = 0
for (let i = 0; i<5; i++){
    alert(i)
}

----------Прирывание цикла (break) и продолжение (continue)
Обычно цикл завершается привычисление условия в значение false. Но мыможем выйти из цикла в любой момент с помощью оператора break
Пример - прога подсчитавает сумму вводимых числе пока пользователь их вводит с клавы

let sum = 0
while (true) {
    let f = +prompt("Введите числа")

    if(!f) break;
    sum = sum + f
}
alert(sum)
Оператор break - ПОЛНОСТЬЮ ПРЕКРАЩАЕТ ВЫПОЛНЕНИЕ ЦИКЛА И ПЕРЕДАЕТ УПРАВ НА БЛОК КОДА ЗА ЕГО ТЕЛОМ.

Оператор continue - При выполнении оператора continue цикл не прирывается а переходит к след иттерации (если условие все еще = true). 
continue используют если понятно что на текущем повторе цикла уже ничего ненужно делать

for (let i = 0; i < 10; i++){
    if (i % 2 === 0) continue;
    alert(i)
}

---------Конструкция switch и case
конструкция switch используютсядля выполнения различ действий в зависимости от знач выражения, она представляет собой альтернативу цепочке if, else, и используется когда надо сравнивать одно знач со могими другими различными вариантами
Пример: с клавиатуры вводится число, 
если число = 3, то выводим результат (мало), 
если число = 4, то выводим результат (в точку),
если число = 5, то выводим результат (много),
если ни один не подходит то выводим (мимо)

let a =+prompt()
switch(a){
    case 3:
        alert("Мало")
        break;

        case 4:
            alert("В точку")
            break;

        case 5:
        alert("Много")
        break;
    default:
        alert("Мимо")
        break;
}
